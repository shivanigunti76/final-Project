1. EC2 INSTANCE 1 (webserver-1)
🔹 Configuration
Name → webserver-1
AMI → Amazon Linux 2
Instance Type → t2.micro
🔹 Security Group
SSH (22) → My IP
HTTP (80) → Anywhere (0.0.0.0/0)
🔹 Commands (VERY IMPORTANT ✅)
-------------------
yum update -y
yum install httpd -y
systemctl start httpd
systemctl enable httpd
echo "This is Server 1" > /var/www/html/index.html
